Import logindb.sql to phpmyadmin and set .env file to connect to this database.

DB_CONNECTION=mysql
DB_HOST=127.0.0.1
DB_PORT=8889 (check phpmyadmin MySQL info)
DB_DATABASE=logindb
DB_USERNAME=root
DB_PASSWORD=root
